Введение

Чтобы включить или отключить расширение, используйте переключатель в правом верхнем углу. Для переключения сайта между черным и белым списками кликните на названии сайта в заголовке.

Вы можете выбрать одну из встроенных цветовых схем в ниспадающем списке на правой панели или установить дополнительные с сайта $website$.

Кроме этого можно настроить выбранную цветовую схему по своему усмотрению. Для этого на левой панели представлены вкладки со всевозможными настройками. Смело меняйте любые параметры - они сразу применяются, но лишь на окно настроек.

Чтобы применить выбранные Вами настройки на текущий сайт нажмите кнопку Применить. Свою цветовую схему можно сохранить для повторного использования на других сайтах.

Если Вы хотите установить текущую цветовую схему в качестве схемы по умолчанию нажмите кнопку В умолчания.

На форуме поддержки Вы можете найти дополнительные инструкции, задать вопрос или предложить свою идею.

Подпишитесь на наши каналы, чтобы следить за обновлениями. Телеграм, ВК.
